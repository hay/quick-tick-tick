# Quick TickTick
The Quick TickTick extension allows you to easily add new tasks to your TickTick without the need to install the official app.

## Commands
This extension has three commands:
- **Add to Inbox** - Adds a task to your inbox
- **Add to List** - Adds a task to a list you configure in settings
- **Add to Any List** - Adds a task to any list you have (this will show a dropdown)

## Installation
After installation you **need** to create an 'app' at [developer.ticktick.com/manage](https://developer.ticktick.com/manage).
1. Go to https://developer.ticktick.com/manage and click 'New App'
2. Enter `Quick TickTick` in the `Name` field and press add
3. Click `Edit` and enter `https://raycast.com/redirect?packageName=quick-tick-tick` in the `OAuth redirect URL` field
4. Now try adding a task to your inbox. You'll get a message indicating you need to enter Client ID and Client Secret
5. Copy Client ID and Client Secret from the App Setting page
6. Done!
